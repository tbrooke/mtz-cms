# Mount Zion Preschool Website Planning

**Date:** January 7, 2026
**Purpose:** Gather requirements for preschool section of mtzcg.com

---

## Overview

We're planning to add a dedicated preschool section to the Mount Zion website. When someone visits **mtzcg.com/preschool**, they'll see ONLY preschool-related content with its own navigation menu, keeping it separate from church content.

**Key Features:**
- Dedicated URL: `mtzcg.com/preschool`
- Separate navigation menu for preschool pages
- Same professional look as church site (or optionally different colors)
- Separate calendar, news/blog, photo galleries
- Easy for preschool staff to update content

**Examples of what visitors would see:**
- Parents researching preschools → `mtzcg.com/preschool` → See programs, admissions info
- Current families → `mtzcg.com/preschool/calendar` → See upcoming events, closures
- Staff → `mtzcg.com/preschool/news` → Post classroom updates
